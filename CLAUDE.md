# Claude Code guide — Art Stockfish

This project keeps its canonical agent guidance in **`AGENTS.md`** (shared with Codex and any
other agent) so the rules never diverge between tools.

**Read `AGENTS.md` first**, then `ART_STOCKFISH_SPEC.md` (what to build) and
`IMPLEMENTATION_PLAN.md` (build order + your task prompt). Everything that governs how to work
here — the design principles, the frozen schema, file-ownership boundaries, the "don't game
gates" rule, and the Loop Contract that defines "done" — lives in those three files.
